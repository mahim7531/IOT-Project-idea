# 1. Introduction

Agriculture plays an important role in the economic development of Bangladesh. Traditional farming methods require continuous manual monitoring of soil conditions, temperature, humidity, and irrigation systems. These methods require more time, human effort, and may cause unnecessary water usage.

The **IoT Based Smart Agriculture Monitoring System** is designed to solve these problems by using **Internet of Things (IoT)** technology. The system collects real-time agricultural information using different sensors and sends the collected data through WiFi communication to a web-based monitoring dashboard.

The system can monitor soil moisture, temperature, and humidity automatically. It can also control irrigation by switching the water pump ON/OFF according to soil moisture conditions.

This project helps farmers reduce manual work, save water resources, improve crop management, and increase agricultural productivity through smart automation.

---

# 2. Problem Statement

Traditional agricultural systems face several challenges:

- Manual checking of soil conditions.
- Lack of real-time monitoring.
- Overwatering or underwatering of crops.
- Difficulty managing farms remotely.
- Wastage of water resources.
- Lack of automated irrigation systems.

To overcome these problems, this project proposes an IoT-based smart agriculture system that can monitor farming conditions automatically and provide efficient irrigation management.

# 6. Proposed System

The proposed system uses an **ESP32 microcontroller** with different sensors to collect agricultural data.

The system components include:

- Soil Moisture Sensor
- DHT11 Temperature & Humidity Sensor
- Relay Module
- Water Pump
- ESP32 Development Board


The collected data is transferred through WiFi to the backend server. The backend stores the information in a database and displays it through an Angular-based web dashboard.

The water pump automatically operates according to soil moisture level.
