# 🌱 Smart Plant Watering System

An IoT-based Smart Plant Watering System that automatically monitors soil moisture levels and waters plants when needed. This project helps save water, reduces manual effort, and keeps plants healthy through automated irrigation.

---

## 📖 Introduction

The Smart Plant Watering System uses a soil moisture sensor to detect the water content in the soil. When the soil becomes dry, the system automatically turns on a water pump to irrigate the plant. Once the soil reaches the desired moisture level, the pump is turned off.

This project is ideal for home gardens, indoor plants, greenhouses, and small-scale agriculture.

## ⚙️ How It Works

1. The soil moisture sensor continuously measures soil moisture.
2. The ESP32 receives the sensor data.
3. If the soil moisture falls below a preset threshold, the relay activates the water pump.
4. Water is supplied to the plant automatically.
5. Once sufficient moisture is detected, the pump stops.
6. Data can be monitored remotely using a mobile application or web dashboard.
