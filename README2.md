## Project Overview

The Smart Attendance System is an IoT-based solution that automates student and employee attendance management using RFID technology. Users simply scan their RFID cards, and attendance records are automatically stored in a cloud database.

This system eliminates manual attendance processes, reduces errors, and provides real-time attendance monitoring through a web dashboard.

---

## Features

- RFID-based attendance marking
- Real-time attendance tracking
- Cloud database integration
- Web dashboard for monitoring
- Automatic date and time recording
- Attendance reports generation
- Secure user identification

---

## System Workflow

1. User scans RFID card.
2. RFID reader detects card ID.
3. ESP32 verifies the ID.
4. Attendance information is uploaded to the cloud.
5. Attendance status is displayed on the dashboard.
6. Reports can be generated by administrators.
