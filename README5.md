
# 🌧️ Smart Automatic Clothes Retracting System

An IoT-based smart clothes drying system that automatically retracts clothes indoors when rain is detected. This project helps protect clothes from getting wet without requiring any manual intervention.

## 📌 Project Description

The Smart Automatic Clothes Retracting System uses a rain sensor and a microcontroller to monitor weather conditions. When rain is detected, a motor automatically pulls the clothesline into a sheltered area. Once the weather becomes clear, the clothesline can be extended back outside.

## 🎯 Objectives

- Detect rainfall automatically.
- Protect clothes from unexpected rain.
- Reduce manual effort.
- Improve convenience using IoT technology.

## ⚙️ Components Used

- Arduino Uno / ESP32
- Rain Sensor Module
- Servo Motor / DC Motor
- L298N Motor Driver
- Power Supply
- Clothesline Mechanism
- Jumper Wires

## 🔄 Working Process

1. The rain sensor continuously monitors weather conditions.
2. When rain is detected, the sensor sends a signal to the microcontroller.
3. The microcontroller processes the signal.
4. The motor is activated automatically.
5. The clothesline retracts into a protected area.
6. After the rain stops, the clothesline can be extended back outside.
