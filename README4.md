# 🤖 IoT Smart Robot

An IoT-based Smart Robot that can be controlled remotely using a smartphone or web application through Wi-Fi. The robot is designed for automation, surveillance, and educational purposes.

## 📖 Project Description

The IoT Smart Robot uses an ESP8266/ESP32 microcontroller to receive commands over the internet and control the movement of motors. It can navigate in different directions, detect obstacles, and provide real-time monitoring.

## ✨ Features

- 🌐 Wi-Fi Controlled Robot
- 📱 Mobile & Web Dashboard Support
- 🚗 Forward, Backward, Left & Right Movement
- 🚧 Obstacle Detection System
- ⚡ Real-Time Response
- 🔋 Battery Powered
- 📡 IoT Connectivity
